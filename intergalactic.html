Intergalactic (LD #30)

My friend and I created a game during a weekend game jam.

During Ludum Dare #30, a game competition that challenges you to create a game from scratch in a weekend, my friend Grant and I created Intergalactic based on the theme "Connected Worlds."

We developed almost every asset and line of code from scratch during that weekend, collecting only audio clips from open source sites - everything else was made by us.

With this project, I wrote the game code in C# using the Unity game engine, while Grant developed the art assets. Below are some screenshots of our game.
